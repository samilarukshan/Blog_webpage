# 🌄 Kandy Travel Webpage

## 🔗 Page Link: https://samilarukshan.github.io/Blog_webpage/

## 📖 Overview

A simple static HTML webpage about **Kandy, Sri Lanka**. This project showcases a travel article with images, text formatting, lists, hyperlinks, and basic CSS styling.

## ✨ Features

* Responsive HTML5 structure
* Basic CSS styling
* Travel article about Kandy
* Image and formatted text
* Ordered list of popular attractions
* External YouTube link

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Jost)

## 📂 Project Structure

```
Kandy-Webpage/
│── index.html
│── s1.jpg
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Make sure `s1.jpg` is in the project folder.
3. Open `index.html` in any modern web browser.

## 📸 Preview

This webpage includes:

* A featured image of Kandy
* Information about the city's history and culture
* A list of top attractions
* A YouTube link for exploring Kandy
